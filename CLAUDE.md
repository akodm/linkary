# Linkary 프로젝트 가이드

## 프로젝트 맥락

- 링크 저장 및 공유 웹 플랫폼
- 반응형 디자인
- PWA 로 브라우저 앱 설치
- SEO 최적화
- 다국어 처리

## 다국어 처리

### 번역 파일 구조

```
src/locales/
├── en.po, en.js      # 영어 (기본)
├── ko.po, ko.js      # 한국어
├── jp.po, jp.js      # 일본어
├── cn.po, cn.js      # 중국어 (간체)
└── es.po, es.js      # 스페인어
```

### 번역 규칙

- **일관성 유지**: 같은 의미의 텍스트는 동일한 번역 사용
- **문화적 적응**: 각 언어권의 문화적 특성 반영

## 주의사항

### 1. 하지 말 것

- any 타입 사용
- 수정이 아닌, 기존 것 참고하여 변환 시 기존 props 의 내용을 누락 혹은 임의로 바꾸지 말 것

### 2. 권장사항

- 접근성 고려
- 재사용성 고려
- 최적화 고려

### 3. 문제 해결 순위

1. 실제 동작 가능하도록 해결책 찾기
2. 기존 코드 일관성 유지
3. 타입 안정성 보장
